ğŸš€ Getting Started

Follow these steps to run Mindbridge locally.

Prerequisites

Node.js installed (v18 or higher)

A Google Cloud API Key for Gemini

Installation

Clone the repository

git clone [https://github.com/yourusername/mindbridge.git](https://github.com/yourusername/mindbridge.git)
cd mindbridge


Navigate to the backend

cd backend


Install dependencies

npm install


Configure Environment
Create a .env file in the backend or root directory and add your API key:

GEMINI_API_KEY=your_actual_api_key_here


Run the Server

npm start


Open in Browser
Visit http://localhost:3000 to start chatting!
